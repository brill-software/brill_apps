Getting Started

A workspace is a set of directories on the server that hold a copy of the repository.

If you wish to make some changes, use your own private workspace.

The development workspace is normally set to contain the develop branch from the repository.

The production system uses the production workspace, which is normally set to the master branch from the repository.

For more help, click on the Help Tab.

To make changes, select your private workspace and click on the SET WORKSPACE button.
